Aplikacijski sloj

Najviši sloj OSI modela, manipulacija podataka na ovom sloju vrši se na način koji korisniku ili softveru omogućuje pristup mreži i njenim uslugama.

Aplikacijski sloj sadrži razne protokole od kojih korisnici najčešće koriste je HTTP (HyperText Transfer Protocol), koji je osnova za World Wide Web (usluga, koju ovaj seminar koristi). Kada preglednik želi web stranicu, on šalje poslužitelju naziv stranice koju želi (klikom na link), poslužitelj zatim šalje stranicu natrag.

Ostali protokoli koji se također koriste, su:

  • File Transfer Protocol (FTP)
  • Trivial File Transfer Protocol (TFTP)
  • Simple Mail Transfer Protocol (SMTP)
  • TELNET
  • DNS (Domain Name System)

Neki gore navedeni protokoli su jednostavnije usluge prijenosa podataka čijom kombinacijom (aplikacijom) nastaju nove usluge u ovisnosti o potrebama korisnika, za koje se radi aplikacija (koja koristi mrežu).
